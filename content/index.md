# このサイトについて

このサイトは気になることを調査したメモ・技術記事・漫画やゲームの感想等を公開するサイトです。

## 記事一覧

公開されている記事はサイドバー or 検索からアクセスができる。

## このサイトについて

技術スタックは以下となる。

- 静的サイトジェネレータ: [Quartz v4](https://quartz.jzhao.xyz/)
- ホスティング: GitHub Pages
- ソース: [Githubリポジトリ](https://github.com/kip2/Knowledge-Is-Power-To-us) 

また、このサイトと同様なサイトを公開するできるテンプレートを作成している。
このサイトもこのテンプレートから作成を行っている。

テンプレートサイト: [サイト作成テンプレート](https://github.com/kip2/publish-site-template)
